>Student Performance Predictor (AI/ML Project)

> Overview

This project is a Machine Learning-based system that predicts a student’s final exam score based on key factors such as study hours, attendance, sleep, and previous performance.

The system simulates an **intelligent agent**:

* Input (Sensors): Student data
* Processing (Model): Machine Learning algorithm
* Output (Actuators): Predicted score



> Problem Statement

Students often do not understand how their daily habits affect their academic performance.
This project aims to solve that by predicting student scores using AI techniques.



> AI/ML Concepts Used

* Artificial Intelligence (AI)
* Machine Learning (Supervised Learning)
* Linear Regression Model
* Intelligent Agent Concept
